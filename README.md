# ithtl-s
后台权限管理系统
本项目是完全基于SpringBoot 1.5、Shiro、Layui、Vue 前后端分离项目，前端使用Nginx服务，Ajax接口/api进行接口转发服务端。

# 目的
学习并使用JAVA框架springboot来开发。
